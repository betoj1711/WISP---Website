# WISP / IRIS Pixel-Accurate Landing Kit V1

This kit uses the approved mockup screenshots directly as the landing page visuals, so the deployed pages match the approved direction instead of drifting.

## Deploy with Cloudflare Pages

Create two Cloudflare Pages projects from the same GitHub repo.

### WISP
- Build command: leave blank
- Build output directory: `wisp-site`
- Custom domain: `wisp-company.com`

### IRIS
- Build command: leave blank
- Build output directory: `iris-site`
- Custom domain: `irisportal.app`

## Notes

This is the fastest way to launch the exact approved visual style. It is not the final coded responsive build. The next step is rebuilding the design into real HTML/CSS components once the visual direction is live and locked.
